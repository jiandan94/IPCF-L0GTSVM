
This is the R code for IPCF-L0GTSVM. 

To run the algorithm code correctly, the **FNN** package must be installed, which can be installed by executing the command `install.packages("FNN")` in R. This package is mainly used for calculating the KNN distances involved in the IPCF weight computation.

Additionally, we provide a demo code named **demo.R** for illustration purposes. The **MASS** package is required for data generation and needs to be installed.
